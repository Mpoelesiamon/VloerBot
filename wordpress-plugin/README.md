# VloerBot WordPress Plugin

This is the WordPress plugin version of the VloerBot chatbot widget. It allows you to easily add the AI-powered chatbot to your WordPress site with a simple plugin installation.

## Features

- ✅ Easy installation via WordPress admin
- ✅ Admin settings page for Supabase configuration
- ✅ Chat widget appears on all pages automatically
- ✅ No coding required
- ✅ Secure credential storage in WordPress database

## Installation

### Step 1: Build the Plugin

First, you need to build the React application for WordPress:

```bash
npm run build:wordpress
```

This will create the `wordpress-plugin/build` directory with all necessary files.

### Step 2: Install the Plugin

1. **Zip the plugin folder:**
   - Navigate to the `wordpress-plugin` directory
   - Create a zip file containing:
     - `vloerbot-chatbot.php` (main plugin file)
     - `build/` directory (with all built assets)

2. **Upload to WordPress:**
   - Go to WordPress Admin → Plugins → Add New
   - Click "Upload Plugin"
   - Select the zip file you created
   - Click "Install Now"
   - Click "Activate Plugin"

3. **Alternative: Manual Installation:**
   - Copy the entire `wordpress-plugin` folder to `/wp-content/plugins/vloerbot-chatbot/`
   - Go to WordPress Admin → Plugins
   - Find "VloerBot Chatbot" and click "Activate"

### Step 3: Configure Supabase Credentials

1. Go to WordPress Admin → Settings → VloerBot
2. Enter your Supabase credentials:
   - **Supabase URL**: Your Supabase project URL (e.g., `https://xxxxx.supabase.co`)
   - **Supabase Publishable Key**: Your Supabase anon/public key
3. Click "Save Settings"

### Step 4: Verify Installation

- Visit any page on your WordPress site
- You should see a chat icon in the bottom-right corner
- Click it to open the chatbot

## How to Find Your Supabase Credentials

1. Go to [supabase.com](https://supabase.com) and sign in
2. Select your project
3. Navigate to **Settings → API**
4. Copy the **Project URL** (this is your Supabase URL)
5. Copy the **anon/public** key (this is your Supabase Publishable Key)

## Development

### Building for WordPress

```bash
npm run build:wordpress
```

This command:
- Builds the React app using the WordPress-specific Vite config
- Outputs files to `wordpress-plugin/build/`
- Creates optimized production files

### Updating the Plugin

After making changes to the React code:

1. Rebuild: `npm run build:wordpress`
2. Re-zip the plugin folder
3. Upload the new zip file to WordPress (or replace files manually)

## Troubleshooting

### Chat widget doesn't appear

1. Check that the plugin is activated
2. Verify Supabase credentials are configured in Settings → VloerBot
3. Check browser console for errors (F12)
4. Ensure the `build` directory exists and contains files

### Supabase connection errors

1. Verify your Supabase URL and key are correct
2. Check that your Supabase project is active
3. Ensure the `chat-uploads` storage bucket exists in Supabase
4. Check browser console for specific error messages

### Build files not found

- Make sure you've run `npm run build:wordpress` before installing
- Verify the `build` directory exists in the plugin folder
- Check file permissions on the server
